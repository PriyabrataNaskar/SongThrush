# SongThrush

![App Icon](https://github.com/PriyabrataNaskar/SongThrush/blob/master/app/src/main/res/mipmap-xxxhdpi/ic_song_thrush.png)

 - Play multiple YouTube Videos
 - Create Your Profile
 - Add your profile image
